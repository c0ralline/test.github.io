## :s 替换
#### 删除1到100行的5到8列
```
:1,100s/\%5c...//
```
\%5c: 匹配第五列，但未选中

...:  匹配三个字符，也就是5，6，7列

#### 删除包含“word”的行
```
:g/word/d
```
g: 匹配指定行

d: 删除

#### 删除不包含“word”的行
```
:g!/word/d
```
g!: 不匹配指定行

d: 删除

## V visual line mode
#### 选中1到4000行
```
1. ESC
2. 1G    //跳转第一行
3. V     //切换模式
4. 4000G //跳转到4000行
```
